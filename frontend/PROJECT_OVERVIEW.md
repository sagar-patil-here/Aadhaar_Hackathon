# Aadhaar DRISHTI — project overview

This repo is **frontend-only** (React dashboard). The Python FastAPI backend, CSV datasets, `requirements.txt`, and `start_backend.sh` have been removed.

## What remains

- React UI: upload flow, dashboard layout, charts (when analysis data exists)
- `start_frontend.sh` — run the dev server

## What was removed

- `backend/` (FastAPI, Tri-Shield analysis, PDF generation, bundled datasets)
- Python dependencies and backend startup script

To run the app: see `README.md` and `SETUP_GUIDE.md`.
