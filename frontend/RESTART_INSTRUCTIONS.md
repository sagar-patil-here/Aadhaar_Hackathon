# Restart instructions

This project no longer includes a backend. Only the React app needs to be restarted.

## Stop the dev server

Press `Ctrl+C` in the terminal where `npm start` is running.

## Start again

```bash6
cd frontend
npm start
```

Open **http://localhost:3000**.
